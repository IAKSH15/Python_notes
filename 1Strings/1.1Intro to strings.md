# 🧵 Python Strings: Indexing, Slicing & Immutability

##  🔢**Indexing in Strings

-  Python strings are indexed starting from **0**.
-  **Forward Indexing**: `(0, 1, 2, 3, ...)`
-  **Reverse Indexing**: `(-1, -2, -3, ...)` (starts from the end)

## 🔒 **Immutability of Strings**

-  Strings in Python are **immutable**.
-  You **cannot change** a string directly.
-  To modify a string, you must **create a new one**.

```python
name = "Hello"

nameshort = name[0:3]   # Gets characters from index 0 to 2 (excludes 3)
print(nameshort)        # Output: Hel

character1 = name[:1]   # Same as name[0:1]
print(character1)       # Output: H

print(name[2:])         # Gets from index 2 to end → Output: llo

character2 = name[-4:-1]  # Slices from index -4 to -2 (excludes -1)
print(character2)         # Output: ell
```

# 📌 **Summary Table**

| Syntax        | Meaning                           | Example Output |
| ------------- | --------------------------------- | -------------- |
| `name[0:3]`   | Characters at index 0 to 2        | `"Hel"`        |
| `name[:1]`    | From start to index 0             | `"H"`          |
| `name[2:]`    | From index 2 to end               | `"llo"`        |
| `name[-4:-1]` | From 2nd char to second-last char | `"ell"`        |
