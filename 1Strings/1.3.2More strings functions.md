# 🧵 Python String Methods : Case, Replace, and Utilities

## 🧪 **Additional Useful String Functions**

```python
text = "hello world"
```

## 🔡 Case Functions

|Function|Description|Example Output|
|---|---|---|
|`text.upper()`|Converts all characters to uppercase|`"HELLO WORLD"`|
|`text.lower()`|Converts all characters to lowercase|`"hello world"`|
|`text.title()`|Capitalizes the first letter of **each word**|`"Hello World"`|
|`text.capitalize()`|Capitalizes only the **first character**|`"Hello world"`|
|`text.swapcase()`|Swaps case of each character|`"HELLO" → "hello"`|

## 🔁 Replacing and Searching

|Function|Description|Example Output|
|---|---|---|
|`text.replace("o", "*")`|Replaces all `"o"` with `"*"`|`"hell* w*rld"`|
|`text.find("world")`|Returns index of first match (`-1` if not found)|`6`|
|`text.count("l")`|Counts number of times a substring occurs|`3`|

## ❓Check Content

|Function|Description|Example Result|
|---|---|---|
|`text.isalpha()`|Checks if all characters are alphabetic|`False`|
|`"Python".isalpha()`|Returns `True`|`True`|
|`text.isnumeric()`|Checks if all characters are digits|`False`|
|`"123".isnumeric()`|Returns `True`|`True`|
|`text.isspace()`|Checks if string only contains spaces|`False`|

## 🧼 Trimming Whitespace

```python
message = "  hello  "
message.strip()       # "hello"
message.lstrip()      # "hello  "
message.rstrip()      # "  hello"
```
