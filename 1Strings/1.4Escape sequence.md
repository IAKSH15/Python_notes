# 🧵 Python Escape Sequences in Strings

## 📘 What Are Escape Sequences?

→ Escape sequences are special characters used in strings to represent things like **newlines**, **tabs**, **quotes**, and **special characters**. They start with a backslash `\`.

## 🧪 Examples with Code and Output

### 📌 `\n` → Inserts a newline

```python
a = "Amit is a college student \nnot a school student" 
print(a)
```

```python
Amit is a college student 
not a school student
```

### 📌 `\"` → Allows use of double quotes inside a string

```python
b = "Amit is a \"good\" boy"    
print(b)
```

```python
Amit is a "good" boy
```

### 📌 `' '` → No escape needed for single quotes if string is enclosed in double quotes

```python
c = "Amit is a 'good' boy"    
print(c)
```

```python
Amit is a 'good' boy
```

### 📌 `\t` → Inserts a tab space

```python
d = "Amit is a college student \tnot a school student"    
print(d)
```

```python
Amit is a college student     not a school student
```

### 📌 `\\` → Prints a single backslash

```python
e = "Amit is a good boy: \\"    
print(e)
```

```python
Amit is a good boy: \
```

### 📌 `\r` → Carriage return: moves cursor to start, so Build overwrites the beginning

```python
f = "Hello World\rBuild"    
print(f)
```

```python
Build World   # Visible output depends on platform
```

### 📌 `\101` → Octal value for `'A'` (octal 101 = decimal 65 = 'A')

```python
g = "Hello World: \101"    
print(g)
```

```python
Hello World: A
```

# 🧠 Summary Table

|Escape Code|Meaning|Example|Output|
|---|---|---|---|
|`\n`|New Line|`"Hi\nBye"`|`Hi``Bye`|
|`\t`|Tab Space|`"Hi\tBye"`|`Hi Bye`|
|`\\`|Backslash|`"\\path"`|`\path`|
|`\"`|Double Quote|`"He said \"Hi\""`|`He said "Hi"`|
|`\'`|Single Quote|`'It\'s fine'`|`It's fine`|
|`\r`|Carriage Return (overwrite)|`"123\rHi"`|`Hi3` or similar|
|`\ooo`|Octal ASCII Code|`\101`|`A`|
