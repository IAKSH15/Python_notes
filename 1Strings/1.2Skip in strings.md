# 🔁 Python String Skipping (Slicing with Steps)

## 📘 Slicing Syntax

```python
string[start:stop:step]
```
### 🧩 Skipping Rules:

1- **If the `start` is omitted**, it defaults to the beginning of the string.
2-  **If the `stop` is omitted**, it defaults to the end of the string.
3-  **If the `step` is omitted**, it defaults to `1` (no skipping).

## 🧪 **Examples:**

```python
a = "amazing"
print(a[0:5:2])   # Output: 'aai'
# Explanation: Characters at index 0, 2, and 4 → 'a', 'a', 'i'

b = "0123456789"
print(b[0:8:3])   # Output: '036'
# Explanation: Characters at index 0, 3, 6 → '0', '3', '6'
```

# 📌 **Summary Table**

| Code       | Meaning                          | Output  |
| ---------- | -------------------------------- | ------- |
| `a[0:5:2]` | From index 0 to 4, skipping by 2 | `'aai'` |
| `b[0:8:3]` | From index 0 to 7, skipping by 3 | `'036'` |

## 🔍 Bonus Tip:

You can reverse a string using a negative step:

```python
text[::-1]  # Reverses the entire string
```
