# 🔤 Python String Functions: len(), startswith(), endswith(), capitalize()

## 🧪 Example Code

```python
name = "Amit"
state = "amit kumar"

print(len(name))                # Output: 4

print(name.endswith("it"))      # Output: True
print(name.endswith("itt"))     # Output: False

print(name.startswith("Am"))    # Output: True
print(name.startswith("am"))    # Output: False (case-sensitive)

print(state.capitalize())       # Output: 'Amit kumar'
```

## 🔍 Function Reference

| Function               | Description                                                        | Example Result                    |
| ---------------------- | ------------------------------------------------------------------ | --------------------------------- |
| `len(string)`          | Returns the **length** (number of characters) in the string        | `len("Amit") → 4`                 |
| `string.endswith(x)`   | Checks if string **ends with** given substring `x`                 | `"Amit".endswith("it") → True`    |
| `string.startswith(x)` | Checks if string **starts with** given substring `x`               | `"Amit".startswith("Am") → True`  |
|                        | 🔸 _Case-sensitive_                                                | `"Amit".startswith("am") → False` |
| `string.capitalize()`  | Converts **first character to uppercase**, all others to lowercase | `"amit kumar" → "Amit kumar"`     |

# ⚠️ Notes:

-  `startswith()` and `endswith()` are **case-sensitive**.
-  `capitalize()` only affects the **first character**, not the whole string.
