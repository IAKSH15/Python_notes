# 🔁 Python Loops – `for` Loop and `while` Loop

## 🔢 Two Types of Loops in Python:

#### _1_ **`for` loop** – Used when the number of iterations is known.
#### _2_ **`while` loop** – Used when looping depends on a condition.

---
### _1_ `for` Loop – Using `range()`

```python
for i in range(1, 6):
    print(i)
```

✅ Loops from **1 to 5** (ending value is **excluded**).

#### 📤 Output:

```python
1
2
3
4
5
```

---
### _2_ `while` Loop – Condition Based

```python
a = 1
while a < 6:
    print(a)
    a += 1
```

✅ Starts at 1 and increases `a` until the condition `a < 6` becomes False.

#### 📤 Output:

```python
1
2
3
4
5
```

| Loop Type    | Syntax Example         | Use Case                            |
