# 🔁 Python `for` Loop with `break` and `continue`

## 🧪 Part 1: Using `break`

```python
for i in range(100):
    if i == 50:
        break  # Stop loop when i equals 50
    print(i)
```

✅ **`break`** is used to **exit the loop immediately** when a condition is met.

#### 📤 Output (first few lines):

```python
0
1
2
...
49
```

🛑 Loop **stops** at `i == 50` and doesn’t print 50 or anything after.

---

## 🧪 Part 2: Using `continue`

```python
for a in range(100):
    if a == 49:
        continue  # Skip the current iteration when a is 49
    print(a)
```

✅ **`continue`** is used to **skip the current iteration** and go to the next one.

#### 📤 Output:

```python
0
1
2
...
48
50
51
...
99
```

📌 `49` is **skipped**, but the loop continues normally.

---

# 🧠 Summary Table

|Statement|Purpose|Behavior Example|
|---|---|---|
|`break`|Exit the loop entirely|Stops printing at `i == 50`|
|`continue`|Skip current loop iteration|Skips `a == 49`, continues with `a == 50`|

---

# 💡 Tips:

-  Use `break` to **exit** early from loops when a condition is met.
-  Use `continue` to **skip** unnecessary work or special cases.
