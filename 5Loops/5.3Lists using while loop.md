# 🔁 Python `while` Loop – Iterate Through a List

## 🧪 Example Code

```python
l = [1, "Hello", "Priya", True, "Subham"]

i = 0
while i < len(l):
    print(l[i])
    i += 1

print(len(l))
```

### ✅ Explanation

-  `l`: A list with mixed data types.
-  `i = 0`: Start index.
-  `while i < len(l)`: Loop runs until `i` reaches the end of the list.
-  `print(l[i])`: Prints the element at the current index.
-  `i += 1`: Moves to the next index.
-  `len(l)`: Returns the total number of elements in the list.

### 📤 Output

```python
1
Hello
Priya
True
Subham
5
```

---
# 🧠 Key Concepts

|Concept|Code|Description|
|---|---|---|
|List Indexing|`l[i]`|Access element by index|
|Length of list|`len(l)`|Total elements in list|
|Loop condition|`i < len(l)`|Ensures loop stops at last item|
|Mixed types|`[1, "Hello", ...]`|Lists can store any data type|
