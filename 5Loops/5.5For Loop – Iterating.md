# 🔁 Python `for` Loop – Iterating Over Data Types

## ✅ What Can You Loop Through in Python?

You can use a `for` loop to iterate through:

-  **Lists**
-  **Tuples**
-  **Strings**

---

### _1_ For Loop with a List

```python
l = [1, 22, 32, 45, 6, 34]

for i in l:
    print(i)
```

#### 📤 Output:

```python
1
22
32
45
6
34
```

---

### _2_ For Loop with a Tuple

```python
t = (1, 8, 9, 55, 45, 87)

for a in t:
    print(a)
```

#### 📤 Output:

```python
1
8
9
55
45
87
```

---

### 🔤 _3_ For Loop with a String

```python
s = "Hello"

for b in s:
    print(b)
```

#### 📤 Output:

```python
H
e
l
l
o
```

---

# 🧠 Summary Table

|Data Type|Loop Code|What It Prints|
|---|---|---|
|List|`for i in l:`|Each element in the list|
|Tuple|`for a in t:`|Each element in the tuple|
|String|`for b in s:`|Each character in the string|
