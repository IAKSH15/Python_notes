# 🔁 Python `while` Loop – Print 1 to 20

## 🧪 Example Code

```python
i = 1
while i < 21:
    print(i)
    i += 1  # or i = i + 1
```

### ✅ Explanation

-  `i = 1`: Starts the counter at 1.
-  `while i < 21`: Runs the loop **as long as** `i` is less than 21.
-  `print(i)`: Prints the current value of `i`.
-  `i += 1`: Increases `i` by 1 each time (same as `i = i + 1`).

---
# 🧠 Tip:

-  Make sure to **update the variable inside the loop**, or it will run infinitely!
-  Use `<= 20` instead of `< 21` for the same result.
