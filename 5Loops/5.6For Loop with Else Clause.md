# 🔁 Python `for` Loop with `else` Clause

## 🧪 Example Code

```python
l = [1, 45, 65, 72]

for item in l:
    print(item)
else:
    print("Done")  # This will print when the for loop finishes normally
```

## ✅ Explanation

-  `for item in l:` loops through each element in the list `l`.
-  The `else` block is executed **only if the loop completes without being interrupted** (e.g. no `break`).
-  This is useful for cleanup or confirmation logic after a loop runs.

#### 📤 Output

```python
1
45
65
72
Done
```

---

## 🧠 When is `else` skipped?

-> If a `break` statement is used and triggered inside the loop, the `else` block is **skipped**.

```python
for item in l:
    if item == 65:
        break
    print(item)
else:
    print("Done")  # This will NOT print
```

#### 📤 Output:

```python
1
45
```

---

# 📌 Summary

| Feature            | Description                                           |
| ------------------ | ----------------------------------------------------- |
| `for...else`       | `else` runs only when loop completes naturally        |
| Skipped on `break` | If loop is interrupted with `break`, `else` won't run |
