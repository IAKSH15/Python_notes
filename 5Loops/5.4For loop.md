# 🔁 Python `for` Loop with `range()` Function

## 🧠 What is `range()`?

-> The `range()` function generates a sequence of numbers, commonly used in `for` loops.

---
## 🧪 Example 1: Basic `range(n)`

```python
for i in range(6):  # range(6) means (0, 1, 2, 3, 4, 5)
    print(i)
```

📌 Starts from 0 by default

#### 📤 Output:

```python
0
1
2
3
4
5
```

---
## 🧪 Example 2: Custom Start, Stop, Step

```python
for a in range(1, 51, 2):  # Start at 1, go up to 50, step by 2
    print(a)
```

📌 Prints **odd numbers** from 1 to 49

#### 📤 Output:

```python
1
3
5
7
...
49
```

---
## 📘 `range(start, stop, step)` Summary

|Argument|Meaning|Example|
|---|---|---|
|`start`|Starting value (inclusive)|`range(5, ...) → 5`|
|`stop`|Ending value (exclusive)|`range(..., 10) → up to 9`|
|`step`|Increment (or decrement)|`range(1, 10, 2)` → 1, 3, 5, 7, 9|

---
# 🧠 Tip:

-  To print **even numbers**: `range(2, 51, 2)`
-  To print in **reverse**: `range(10, 0, -1)`
