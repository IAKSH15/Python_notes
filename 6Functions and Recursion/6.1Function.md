# 🧮 Python Functions – Introduction & Example

## ✅ What is a Function?

-  A **function** is a group of related statements that performs a **specific task**.
-  Functions help in **code reuse**: they can be **called multiple times** from different parts of the program.

---

## 🧪 Example: Function to Calculate Average

```python
def avg():
    a = int(input("Enter a number: "))
    b = int(input("Enter a number: "))
    c = int(input("Enter a number: "))
    avg = (a + b + c) / 3
    print("The average of the three numbers is:", avg)

avg()  # Function call
```

### ✅ Explanation

|Part|Description|
|---|---|
|`def avg():`|Function definition named `avg`|
|`input()`|Takes 3 numbers from the user|
|`(a + b + c)/3`|Calculates average|
|`print(...)`|Displays the result|
|`avg()`|Calls the function|

#### 📤 Sample Output

```python
Enter a number: 10
Enter a number: 20
Enter a number: 30
The average of the three numbers is: 20.0
```

---

# 🧠 Tips:

-  You can define **multiple functions** in one program.
-  Use **parameters** to make functions more flexible (e.g., `def avg(a, b, c):`).
-  Python allows **default values**, **return values**, and **docstrings**.
