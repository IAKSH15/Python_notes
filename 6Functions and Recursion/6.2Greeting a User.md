# 👋 Python Function – Greeting a User

## ✅ Objective:

Write a program to greet a user with `"Good Day"` using a function.

---

## 🧪 Example Code

```python
def goodDay():
    print("Good Day")

goodDay()
```

### ✅ Explanation

|Code Part|Description|
|---|---|
|`def goodDay():`|Defines a function named `goodDay`|
|`print("Good Day")`|Action: prints greeting when called|
|`goodDay()`|Calls the function to display the greeting|

#### 📤 Output

```python
Good Day
```

---

# 🧠 Tip:

You can enhance it with a name:

```python
def goodDay(name):
    print("Good Day,", name)

goodDay("Priya")
```

#### 📤 Output:

```python
Good Day, Priya
```
