# 🧮 Python Function – With Parameters and Return Value

## ✅ Objective:

-> Create a function that greets the user with a name and prints a custom closing message, then returns a value.

---

## 🧪 Example Code

```python
def goodDay(name, ending):
    print("Good Day, " + name)
    print(ending)
    return "Done"  # Returns a value to the caller

a = goodDay("Amit", "Thank You")
print(a)
```

### ✅ Explanation

|Code Part|Description|
|---|---|
|`def goodDay(name, ending):`|Function with two parameters: `name`, `ending`|
|`print("Good Day, "+name)`|Greets the user by name|
|`print(ending)`|Prints the custom closing message|
|`return "Done"`|Sends `"Done"` back to where the function was called|
|`a = goodDay(...)`|Stores the return value in variable `a`|
|`print(a)`|Prints the returned value (`"Done"`)|

#### 📤 Output

```python
Good Day, Amit
Thank You
Done
```

---

# 🧠 Why Use `return`?

-  Allows a function to **send a result** back to the caller.
-  Useful for further computation or logging.
