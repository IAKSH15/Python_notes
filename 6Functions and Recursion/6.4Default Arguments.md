# 🧮 Python Function – Default Arguments

## ✅ Objective:

-> Create a function that uses a **default parameter** so you can choose whether or not to provide all arguments during the function call.

---

## 🧪 Example Code

```python
def goodDay(name, ending="Thank you"):
    print(f"Good Day, {name}")
    print(ending)

goodDay("Amit")
goodDay("Sonu", "Thanks")
```

### ✅ Explanation

|Code Part|Description|
|---|---|
|`ending="Thank you"`|`ending` is a **default argument**|
|`goodDay("Amit")`|Uses the **default** value for `ending`|
|`goodDay("Sonu", "Thanks")`|Overrides the default and uses `"Thanks"`|

#### 📤 Output

```python
Good Day, Amit
Thank you
Good Day, Sonu
Thanks
```

---

# 🧠 Key Notes on Default Arguments

-  Default arguments **must come after** non-default ones.
-  If a value is provided, it **overwrites the default**.
-  If no value is provided, Python uses the **default**.

---

## 📌 Function Header Rule:

```python
# ✅ Valid:
def greet(name, message="Hi"):
    ...

# ❌ Invalid (non-default after default):
def greet(message="Hi", name):  # ❌ SyntaxError
    ...
```
