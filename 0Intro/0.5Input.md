#Python 

-  In Python, `input()` is a **built-in function** used to take user input from the keyboard.

```python
a = int(input("Enter the value of a: "))  # Converts input to an integer
#  a = input("Enter the value of a: ")    This will store the input as a string
b = int(input("Enter the value of b: "))  # Converts input to an integer
# b = input("Enter the value of b: ")     This will store the input as a string

print("The value of a is:", a)
print("The value of b is:", b)
print("The sum of a and b is:", a + b)
```

# 🔍 Key Points:

-  input()` takes input as a **string** by default.
-  Using `int()` converts the input into an **integer**, so math operations can be performed.
-  Without `int()`, addition (`a + b`) would **join strings**, not add numbers (e.g., `"2" + "3"` becomes `"23"`).
