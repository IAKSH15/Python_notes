#Python  

- You can use the `type()` function to check the data type of any variable.

```python
a = 23
t = type(a)
print(t)      # <class 'int'>

b = "Hello"
x = type(b)
print(x)      # <class 'str'>

c = "3.14"
d = float(c)    # Converts string to float
y = type(d)
print(y)      # <class 'float'>

```

# Common Type Conversions

## _1 Integer to String

```python
e = 5
print(type(e))    # <class 'int'>

f = str(e)       # Convert int to string
print(type(f))    # <class 'str'>
```

## _2 Integer to Float

```python
g = float(e)
print(type(g))   # <class 'float'>
```

## _3 String to Integer

```python
h = "23"
print(type(h))    # <class 'str'>

i = int(h)        # Convert string to int
print(type(i))    # <class 'int'>
```

## _4 Summary

 - Use `type(variable)` to check the data type.
 - Use `str()`, `int()`, and `float()` to convert between types.
 - Make sure the string is **numerical** when converting to `int` or `float`.
