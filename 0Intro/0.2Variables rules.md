#Python 

-  A variable name can contain alphabets, digits and underscores.
-  A variable name can only start with an alphabets or underscores.
-  A variable name cannot start with a digit.
-  A variable name cannot contain any special characters except underscores or have space between them.

```python
a = 12  
aaa = 344  
hello = 23  
_hello = 14  
hello23 = 45
# @hello = 23   # Invalid due to @ symbol.  
# s@meer = 45   # Invalid due to @ symbol.  
# 12 = 34       # Invalid due to variable name starting with a digit.
```
