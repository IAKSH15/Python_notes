#Python  

# Arithmetic Operators
```python
a = 2
b = 4
c = a+b
print(c)
print(b/a)       # This will print the result with decimal.
print(b//a)      # This will print the result without decimal.
```

# Assignment Operators
```python
a = 4-2    # Assign 4-2 in a
print(a)  
b = 5
b -=3   # Decrement the value of b by 3 and then assign it to b
print(b)
```

# Comparison Operators
```python
d = 5==5
print(d)
f = 5<7
print(f)
```

# Logical Operators
```python
# Truth table of 'or'

print("True or False is",True or False)
print("False or False is",False or False)
print("True or True is",True or True)
print("False or True is",False or True)

# Truth table of 'and'

print("True and False is",True and False)
print("False and False is",False and False)
print("True and True is",True and True)
print("False and True is",False and True)
```

# Not Operator
- Not operator is used to negate the Boolean value means it will make "true - false" and "false - true"
```python
print(not(False))
print(not(True))
```
