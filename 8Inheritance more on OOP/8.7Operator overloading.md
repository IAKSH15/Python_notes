# ➕ Python OOP – Operator Overloading with Magic Methods

## ✅ What is Operator Overloading?

-> Operator overloading allows custom behavior when using **operators** (`+`, `-`, `*`, `/`, `//`, etc.) with user-defined objects.  
This is done using **special methods** like `__add__`, `__sub__`, etc.

---

## 🧪 Example: Overloading `+` Operator

```python
class Number:
    def __init__(self, n):
        self.n = n

    def __add__(self, num):  # Overloads the `+` operator
        return self.n + num.n

p = Number(5)
u = Number(6)

print(p + u)
```

### 📤 Output

```python
11
```

## ✅ Other Common Magic Methods for Arithmetic

```python
def __sub__(self, num):         # For -
    return self.n - num.n

def __mul__(self, num):         # For *
    return self.n * num.n

def __truediv__(self, num):     # For /
    return self.n / num.n

def __floordiv__(self, num):    # For //
    return self.n // num.n
```

---

## 🧪 Example with All Arithmetic Overloads

```python
class Number:
    def __init__(self, n):
        self.n = n

    def __add__(self, num):
        return self.n + num.n

    def __sub__(self, num):
        return self.n - num.n

    def __mul__(self, num):
        return self.n * num.n

    def __truediv__(self, num):
        return self.n / num.n

    def __floordiv__(self, num):
        return self.n // num.n

x = Number(10)
y = Number(3)

print(x + y)   # 13
print(x - y)   # 7
print(x * y)   # 30
print(x / y)   # 3.333...
print(x // y)  # 3
```
