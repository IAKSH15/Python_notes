# 🧬 Python OOP – Inheritance

## ✅ What is Inheritance?

-> **Inheritance** allows a class (child/derived class) to **inherit methods and properties** from another class (parent/base class), enabling **code reuse and extension**.

---

## 🧪 Example Code

```python
class Employee:
    company = "ITC"

    def show(self, name, salary):
        self.name = name
        self.salary = salary
        print(f"The name of the Employee is {self.name} and the salary is {self.salary}")

class Programmer(Employee):  # Inherits from Employee
    company = "ITC Infotech"

    def showLanguage(self):
        print(f"The name is {self.name} and he is good in programming language {self.language} language")

a = Employee()
a.show("Rohan", 123000)

b = Programmer()
print(a.company, b.company)
```

### 📤 Output

```python
The name of the Employee is Rohan and the salary is 123000
ITC ITC Infotech
```

---

## ✅ Explanation

|Class/Method|Role|
|---|---|
|`Employee`|**Base class** with `company` attribute and `show()` method|
|`Programmer(Employee)`|**Derived class** inheriting from `Employee`|
|`a.show(...)`|Calls method from the base class|
|`b.company`|Accesses class attribute overridden in `Programmer`|

---

## 🧠 Key Inheritance Concepts

|Concept|Behavior|
|---|---|
|Inherits methods|`Programmer` inherits `show()` from `Employee`|
|Attribute override|`Programmer.company` overrides `Employee.company`|
|Method extension|`showLanguage()` adds new functionality in the child class|

---

## 📌 Inheritance Flow Diagram

```python
Employee
   ↓
Programmer (inherits show(), company overridden, adds showLanguage())
```

---

## ⚠️ Reminder:

-> You must call `show()` first **before** `showLanguage()` if you want to use `self.name`, or initialize it via `__init__()`.
