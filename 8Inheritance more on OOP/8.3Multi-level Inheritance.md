# 🧬 Python OOP – Multi-level Inheritance

## ✅ Concept:

-> In **multi-level inheritance**, a class inherits from a derived class, which in turn inherits from another base class forming a **chain of inheritance**.

---

## 🧪 Example Code

```python
class Employee:
    a = 2

class Programmer(Employee):
    b = 3

class Manager(Programmer):
    c = 4

# Instance of Employee
o = Employee()
print(o.a)         # ✅ Accesses a from Employee
# print(o.b)       # ❌ Error: 'Employee' has no attribute 'b'

# Instance of Programmer
o = Programmer()
print(o.a, o.b)    # ✅ Inherits 'a' from Employee, 'b' from Programmer

# Instance of Manager
o = Manager()
print(o.a, o.b, o.c)  # ✅ Inherits from Programmer → Employee
```

### 📤 Output

```python
2
2 3
2 3 4
```

---

```python
Employee
   ↓
Programmer
   ↓
Manager
```

-  Each class inherits all attributes from its **ancestors**.

---

## ✅ Explanation Table

|Class|Attributes Available|Inherited From|
|---|---|---|
|`Employee`|`a`|Base class|
|`Programmer`|`a`, `b`|Inherits from `Employee`|
|`Manager`|`a`, `b`, `c`|Inherits from `Programmer` → `Employee`|

---

# ⚠️ Note:

-> Trying to access an attribute not in the current or inherited classes will raise:

```python
AttributeError: 'ClassName' object has no attribute 'X'
```
