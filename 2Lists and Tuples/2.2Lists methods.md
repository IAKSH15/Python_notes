# 🧾 Python List Methods: Append, Sort, Insert, Pop, Remove

## 📘 Original List

```python
name = ["Mango", "Amit", 33.4, 33, False]
print(name)
```

```python
['Mango', 'Amit', 33.4, 33, False]
```

### _1_ `append()`

```python
name.append("Rohan")
print(name)
```

📌 Adds a value to the **end** of the list.  
✅ List grows by 1 element.

```python
['Mango', 'Amit', 33.4, 33, False, 'Rohan']
```

## 📘 Another List for Testing

```python
l1 = [1, 22, 44, 5, 77, 9, 2]
```

### _2_ remove(44)

```python
l1.remove(44)
print(l1)
```

📌 Removes the **first occurrence** of `44`.  
✅ Does not return the removed item.

```python
[1, 22, 5, 77, 9, 2]
```

### _3_ insert(2, 5)

```python
l1.insert(2, 5)
print(l1)
```

📌 Inserts the value `5` at **index 2**.  
✅ Shifts the rest of the elements to the right.

```python
[1, 22, 5, 5, 77, 9, 2]
```

### _4_ pop(2)

```python
value = l1.pop(2)
print(value)
print(l1)
```

📌 Removes the value at index `2` and returns it.  
✅ Useful for extracting and deleting together.

```python
5
[1, 22, 5, 77, 9, 2]
```

### _5_ sort()

```python
l1.sort()
print(l1)
```

📌 Sorts the list in **ascending order**.  
✅ All items must be of comparable types (e.g., all numbers).

```python
[1, 2, 5, 9, 22, 77]
```

### _6_ reverse()

```python
l1.reverse()
print(l1)
```

📌 Reverses the list **in-place**.  
✅ Order is exactly flipped.

```python
[77, 22, 9, 5, 2, 1]
```

# 📝 Summary Table – With Code, Notes, and Output

|Method|Description|Code Example|Output Example|
|---|---|---|---|
|`.append(x)`|Adds `x` to the end|`["A", "B"].append("C")`|`["A", "B", "C"]`|
|`.remove(x)`|Removes the first occurrence of `x`|`[1, 2, 2].remove(2)`|`[1, 2]`|
|`.insert(i, x)`|📌 Inserts `x` at index `i` ✅ Shifts right|`[1, 3].insert(1, 2)`|`[1, 2, 3]`|
|`.pop(i)`|Removes and returns item at index `i`|`x = [10, 20, 30].pop(1)`|`x = 20`, `[10, 30]`|
|`.sort()`|Sorts the list in ascending order|`[3, 1, 2].sort()`|`[1, 2, 3]`|
|`.reverse()`|Reverses the list|`[1, 2, 3].reverse()`|`[3, 2, 1]`|
