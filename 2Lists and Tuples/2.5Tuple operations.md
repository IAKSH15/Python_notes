# 📦 Python Tuples – Advanced Operations

### 🔗 Tuple Concatenation

```python
t1 = (1, 2, 3, 4)
t2 = (5, 6, 7, 8)

print(t1 + t2)
```

📌 Combines two tuples into one.

```python
(1, 2, 3, 4, 5, 6, 7, 8)
```

### 🔁 Tuple Repetition

```python
print(t1 * 2)
```

📌 Repeats the tuple elements twice.

```python
(1, 2, 3, 4, 1, 2, 3, 4)
```

### 📏 Finding Length

```python
print(len(t2))
```

📌 Returns number of elements in `t2`.

```python
4
```

### 🔍 Membership Checking

```python
print(4 in t2)          # False
print(12 not in t1)     # True
print(1 in t1)          # True
```

```python
False
True
True
```

## 📦 Tuple Packing and Unpacking

### 📦 Packing

```python
person = ("John", 25, "Software Engineer")
```

📌 Group multiple values into a single tuple.

## 📤 Unpacking

```python
name, age, profession = person

print(name)
print(age)
print(profession)
```

```python
John
25
Software Engineer
```

## 🧪 Partial Unpacking with `*`

```python
numbers = (1, 2, 3, 4, 5, 6)
a, *b, c, d = numbers

print(a)
print(b)
print(c)
print(d)
```

📌 `*b` captures the middle part.

```python
1
[2, 3, 4]
5
6
```

## 🗺️ Using Tuples as Dictionary Keys

```python
coordinates = {
    (12, 3): "point A",
    (34, 6): "point B"
}

print(coordinates[(12, 3)])
```

📌 Tuples are immutable, so they can be used as keys in dictionaries.

```python
point A
```
