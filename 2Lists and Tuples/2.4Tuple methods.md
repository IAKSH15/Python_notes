# 📦 Python Tuples – Useful Methods & Conversions

## 🧪 Tuple Example

```python
a = (1, 45, "hello", 43, 1, False)

print(a)
print(type(a))
```

#### 📤 Output:

```python
(1, 45, 'hello', 43, 1, False)
<class 'tuple'>
```

### _1_ 🔢 `count()` Method

```python
no = a.count(1)
print(no)
```

📌 Counts how many times the value `1` appears in the tuple.  
✅ `True` is considered equal to `1` in Python, so it may affect the count.

```python
3
```

### _2_ 📍 `index()` Method

```python
no2 = a.index(43)
print(no2)
```

📌 Returns the **first index** of the value `43`.  
✅ Raises an error if the value is not found.

```python
3
```

### _3_ 🔁 List ↔ Tuple Conversions

```python
b = tuple([1, 3, 44, "Hello"])
print(type(b))
```

```python
<class 'tuple'>
```

### _4_ ✅ Converting a Tuple to List

```python
c = list(a)
print(c)
```

```python
[1, 45, 'hello', 43, 1, False]
```

# 🧠 Summary Table

|Function / Action|Description|Example|Output|
|---|---|---|---|
|`a.count(x)`|Counts how many times `x` occurs|`a.count(1)`|`3`|
|`a.index(x)`|Returns index of first `x`|`a.index(43)`|`3`|
|`tuple(list)`|Converts list to tuple|`tuple([1, 2])`|`(1, 2)`|
|`list(tuple)`|Converts tuple to list|`list((1, 2))`|`[1, 2]`
