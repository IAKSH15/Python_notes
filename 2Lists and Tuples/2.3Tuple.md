# 📦 Python Tuples – Immutable Collections

## ✅ What is a Tuple?

-> A **tuple** is a collection of **ordered**, **immutable** items in Python. It can contain elements of different data types (strings, integers, Booleans, etc.).

## 🧪 Tuple Examples

```python
# Creating a tuple with multiple elements
a = (1, 45, "hello", 5.5, False)

print(a)
print(type(a))
```

#### 📤 Output:

```python
(1, 45, 'hello', 5.5, False)
<class 'tuple'>
```

## 🧠 Important Notes

### 🟢 Creating Special Tuples:

```python
a = (1,)      # ✅ Single-element tuple (Note the comma!)
a = ()        # ✅ Empty tuple
```

### 🔴 Tuples are immutable:

```python
a[0] = 22     # ❌ This will cause an error!
```

-> **📌 You cannot change, add, or remove elements once a tuple is created.**

# 🔁 Tuple vs List Comparison

|Feature|Tuple| List                     |
| -------- | ---------------------- | ------------------------ |
|Syntax|`a = (1, 2)`| `a = [1, 2]`             |
|Mutable|❌ No (Immutable)| ✅ Yes                    |
|Ordered|✅ Yes| ✅ Yes                    |
|Use Case|Fixed data collections| Dynamic, changeable data
