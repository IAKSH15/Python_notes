# ✅ What is a List in Python?

-> A **list** in Python is a **collection of items** (like numbers, strings, or other data) stored in a **single variable**. It is **mutable**, meaning you can **change, add, or remove** items.

# 📚 Python Lists: Indexing & Mutability

## 🧪 Sample Code

```python
name = ["Mango", "Amit", 33.4, 33, False]

print(name[0])          # Output: Mango

name[0] = "Apple"       # Lists are mutable, so we can change elements

print(name[0])          # Output: Apple

print(name[1:4])        # Output: ['Amit', 33.4, 33]
```

## 🧠 Key Concepts

### 🔄 Mutability

-  Lists in Python are **mutable**, meaning you can **change**, **update**, or **delete** their elements.
-  Unlike **strings**, which are **immutable**, lists allow direct assignment:

```python
 name[0] = "Apple"
```

## 📌 Indexing & Slicing Recap

|Operation|Result|
|---|---|
|`name[0]`|First element → `"Mango"`|
|`name[1:4]`|Slice from index 1 to 3 → `["Amit", 33.4, 33]`|
|`name[-1]`|Last element → `False`|

|Feature|List|String|
