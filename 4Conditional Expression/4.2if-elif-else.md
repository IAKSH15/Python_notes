# 🧮 Python Conditional Statement – `if-elif-else`

## 🧪 Example Code

```python
a = int(input("Enter your age: "))

if a >= 18:
    print("You are above the age of consent")
elif a < 0:
    print("You are entering an invalid negative age")
elif a == 0:
    print("You are entering 0 age which is not valid")
else:
    print("You are below the age of consent")

print("End of code")
```

## ✅ Explanation

-  `if` checks if age is 18 or more.
-  `elif a < 0` checks for **invalid negative ages**.
-  `elif a == 0` catches **zero input**, which is also invalid.
-  `else` runs if age is between **1 and 17** (below consent age).
-  The last `print("End of code")` runs regardless of conditions.

# 💡 Pro Tip:

The order of `elif` matters:
-  Place **specific** conditions (like `a < 0`) before general ones (like `a < 18`).
-  Otherwise, specific checks may be **skipped**.
