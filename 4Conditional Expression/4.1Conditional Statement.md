# 🧮 Python Conditional Statement – `if-else`

## 🧪 Example Code

```python
a = int(input("Enter your age: "))

if a >= 18:
    print("You are above the age of consent")
else:
    print("You are below the age of consent")

print("End of code")
```

## ✅ Explanation

-  `input()` takes user input as a string.
-  `int()` converts it to an integer.
-  `if a >= 18`: checks if age is **18 or older**.
-  `else`: runs if condition is **False**.
-  The final `print("End of code")` runs regardless of condition.

### 📤 Sample Output 1 (if input is `20`)

```python
Enter your age: 20
You are above the age of consent
End of code
```

## 📤 Sample Output 2 (if input is `16`)

```python
Enter your age: 16
You are below the age of consent
End of code
```

# 🧠 Tip:

-> Always indent the code block inside `if` and `else` by **4 spaces** or **1 tab** in Python.
