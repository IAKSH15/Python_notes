# 🧮 Python Conditional Logic – Multiple `if` Statements

## 🧪 Example Code

```python
a = int(input("Enter your age: "))

# if statement no. 1
if a % 2 == 0:
    print("Entered value is even")
# End of if statement no. 1

# if statement no. 2
if a >= 18:
    print("You are above the age of consent")
elif a < 0:
    print("You are entering an invalid negative age")
elif a == 0:
    print("You are entering 0 age which is not valid")        
else:
    print("You are below the age of consent")

print("End of code")
```

## ✅ Explanation

#### 🔹 `if` Statement 1: Even Number Check
-  Checks if the entered value is **even** using the modulo operator: `a % 2 == 0`.
### 🔹 `if-elif-else` Block: Age Validation
 `a >= 18`: Legal adult
-  `a < 0`: Invalid negative age
-  `a == 0`: Zero is not valid as an age
-  `else`: For ages between **1 and 17**

# 🧠 Key Takeaways

-  You can use **multiple `if` statements** independently.
-  `if-elif-else` blocks are **exclusive**: only one branch will execute.
-  Use `a % 2 == 0` to check for **even numbers**.
