# 📘 Python Dictionary – Methods and Operations

## ✅ Creating Dictionaries

```python
a = {}  # Empty dictionary
marks = {
    "Amit": 90,
    "Jhon": 85,
    "Kunal": 86,
    0: "Rohan"  # Integer key is allowed
}
```

## 🔍 Dictionary Methods

### _1_ items()

```python
print(marks.items())
```

📌 Returns all key-value pairs as a view of **tuples**.

```python
dict_items([('Amit', 90), ('Jhon', 85), ('Kunal', 86), (0, 'Rohan')])
```

### _2_ keys()

```python
print(marks.keys())
```

📌 Returns all the **keys** in the dictionary.

```python
dict_keys(['Amit', 'Jhon', 'Kunal', 0])
```

### _3_ values()

```python
print(marks.values())
```

📌 Returns all the **values** in the dictionary.

```python
dict_values([90, 85, 86, 'Rohan'])
```

### _4_ update()

```python
marks.update({"Amit": 95, "Dhruv": 75})
print(marks)
```

📌 Updates existing key `"Amit"` and adds a new key `"Dhruv"`.

```python
{'Amit': 95, 'Jhon': 85, 'Kunal': 86, 0: 'Rohan', 'Dhruv': 75}
```

### _5_ get()

```python
print(marks.get("Rohit"))  # Key doesn't exist
print(marks.get("Kunal"))  # Key exists
```

📌 Safely retrieves value of a key.  
✅ Returns `None` if the key doesn’t exist, instead of raising an error.

```python
None
86
```

### _6_ Access using brackets `[]`

```python
print(marks["Amit"])
```

📌 Returns the value directly.  
⚠️ Raises `KeyError` if the key doesn’t exist.

```python
95
```

### _7_ len()

```python
print(len(marks))
```

📌 Returns total number of key-value pairs.

```python
5
```

# 📊 Summary Table of Dictionary Methods

|Method|Description|Example|Output Example|
|---|---|---|---|
|`.items()`|Returns key-value pairs as tuples|`d.items()`|`[('a',1), ('b',2)]`|
|`.keys()`|Returns all keys|`d.keys()`|`['a', 'b']`|
|`.values()`|Returns all values|`d.values()`|`[1, 2]`|
|`.update({...})`|Updates or adds key-value pairs|`d.update({"x": 3})`|Adds/updates `x: 3`|
|`.get(key)`|Safely gets value or returns `None` if not found|`d.get("x")`|Value or `None`|
|`d[key]`|Access value (raises error if key is missing)|`d["a"]`|Value or `KeyError`|
|`len(d)`|Returns number of items in dictionary|`len(d)`|`n` (number of key-value pairs)|
