# 🧮 Python Sets – Set Operations

## ✅ Set Definitions

```python
s1 = {1, 2, 4, 3, 45}
s2 = {5, 6, 7, 2, 45}
```

## `union()` – Combine All Unique Elements

```python
print(s1.union(s2))
```

📌 Combines elements from **both sets**, keeping **only unique** values.  
✅ Does **not** duplicate common elements.

```python
{1, 2, 3, 4, 5, 6, 7, 45}
```

## `intersection()` – Common Elements

```python
print(s1.intersection(s2))
```

📌 Returns only the **elements that exist in both** `s1` and `s2`.

```python
{2, 45}
```

## `s1 - s2` – Difference: Elements in `s1` not in `s2`

```python
print(s1 - s2)
```

📌 Returns elements that are in `s1` but **not in** `s2`.

```python
{1, 3, 4}
```

## `s2 - s1` – Difference: Elements in `s2` not in `s1`

```python
print(s2 - s1)
```

📌 Returns elements that are in `s2` but **not in** `s1`.

```python
{5, 6, 7}
```

# 🧠 Summary Table

|Operation|Description|Example Output|
|---|---|---|
|`s1.union(s2)`|Combines both sets (no duplicates)|`{1, 2, 3, 4, 5, 6, 7, 45}`|
|`s1.intersection(s2)`|Common elements in both sets|`{2, 45}`|
|`s1 - s2`|Elements in `s1` but not in `s2`|`{1, 3, 4}`|
|`s2 - s1`|Elements in `s2` but not in `s1`|`{5, 6, 7}`|
