# 🧺 Python Sets – Methods and Operations

## 🧪 Initial Code

```python
a = {2, 4, "hello", 55}
print(type(a))
```

### 📤 Output:

```python
<class 'set'>
```

## 🔧 Set Methods

### _1_ `add()` – Add an Element

```python
a.add(90)
```

📌 Adds `90` to the set.  
✅ No error if element already exists (won’t be duplicated).

### _2_ `remove()` – Remove Specific Element

```python
a.remove("hello")
```

📌 Removes `"hello"` from the set.  
⚠️ Raises `KeyError` if the item is not present.

### _3_ `pop()` – Remove and Return a Random Element

```python
# a.pop()
```

📌 Removes a **random element** (because sets are unordered).  
✅ Use with caution if you rely on element order.

### _4_ `clear()` – Remove All Elements

```python
# a.clear()
```

📌 Empties the entire set.  
✅ Set becomes `{}` (but still a set type).

## 📤 Final Output

```python
print(a)
```

```python
{2, 4, 90, 55}
```

# 📝 Summary Table of Set Methods

|Method|Description|Example|Notes|
|---|---|---|---|
|`add(x)`|Adds `x` to the set|`a.add(10)`|✅ Adds if not already present|
|`remove(x)`|Removes `x` from set|`a.remove("hi")`|❌ Error if `x` not found|
|`pop()`|Removes a random element|`a.pop()`|⚠️ No index control|
|`clear()`|Removes all elements|`a.clear()`|Set becomes empty|
