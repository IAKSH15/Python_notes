# 📘 Python Dictionary – Key-Value Pairs

## ✅ Definition:

A **dictionary** in Python is a **collection of key-value pairs**.  
It is **mutable**, which means:

- You can **change values**
- You can **add new key-value pairs**
- Keys must be **unique** and **immutable**

## 🧪 Example Code

```python
marks = {
    "Amit": 90,
    "Jhon": 85,
    "Kunal": 86,
}

print(marks["Amit"], type(marks))
print(marks["Kunal"], marks["Jhon"])
```

### 📤 Output

```python
90 <class 'dict'>
86 85
```

# 🔍 Key Concepts

| Feature               | Description                                | Example                |
| --------------------- | ------------------------------------------ | ---------------------- |
| Access value          | Use the key inside square brackets         | `marks["Amit"]` → `90` |
| Check type            | `type(marks)` → confirms it's a dictionary | `<class 'dict'>`       |
| Dictionary is mutable | You can change or add values               | `marks["Amit"] = 95`   |
