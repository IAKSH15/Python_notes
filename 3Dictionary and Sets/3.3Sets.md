# 🧺 Python Sets – Properties and Basics

## ✅ Definition:

-> A **set** in Python is an **unordered**, **unindexed** collection of **unique** and **immutable elements** (though the set itself is mutable).  
It is useful when you want to store **distinct items only**.

## 🧪 Key Properties of Sets

-  ❌ **Unordered** → No guaranteed order of elements
-  ❌ **Unindexed** → You **cannot** access elements using indexes
-  ✅ **Mutable** → You can **add/remove** elements
-  ❌ **No duplicates** → All elements must be **unique**

## Creating Sets

```python
b = set()          # Creates an empty set
a = {22, 3, 5, "Hello"}
c = {2, 1, 5, 6, 2, 3, 2}
```

#### 📌 Note:

```python
b = {}  # ❌ Creates a dictionary, not a set!
```

## 🧪 Printing and Length

```python
print(c)
print(len(a))
print(len(c))
```

#### 📤 Output:

```python
{1, 2, 3, 5, 6}     # Repeated elements like 2 are automatically removed
4                  # Length of set `a`
5                  # Length of set `c`
```

# 📌 Summary Table

|Feature|Description|Example|
|---|---|---|
|**Unordered**|No guarantee of element order|`print(set([3, 1, 2])) → {1,2,3}`|
|**Unindexed**|No index access (e.g., `set[0]` ❌)|Not allowed|
|**Mutable**|You can add/remove elements|`set.add(4)`|
|**Unique values only**|Duplicates are removed automatically|`{1, 1, 2} → {1, 2}`|
|`set()`|Creates an empty set|`set()`|
|`{}`|Creates an empty dictionary, not a set|`type({}) → <class 'dict'>`|
